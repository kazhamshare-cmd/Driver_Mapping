# 店舗スタッフアプリ (Shop Staff App)

予約管理プラットフォーム - 店舗スタッフ用モバイルアプリ

## 📱 概要

Raspberry Pi不要！スマホ1台で注文管理・レジ・予約管理・プリンター印刷ができるアプリです。

## ✨ 主な機能

1. **ログイン** - スタッフ認証
2. **代理注文** - 口頭注文をテーブル別に受付
3. **注文管理** - リアルタイム注文一覧とステータス更新
4. **レジ・会計** - 会計処理とレシート印刷
5. **予約管理** - 予約の確認・承認
6. **プリンター設定** - Bluetoothプリンター接続と自動印刷設定

## 🖨️ 対応プリンター

- Bluetooth ESC/POSプリンター（エプソン TM-m30など）
- 自動キッチン伝票印刷
- 自動レシート印刷

## 💰 コスト

- Raspberry Pi不要
- スマホ + Bluetoothプリンター のみ
