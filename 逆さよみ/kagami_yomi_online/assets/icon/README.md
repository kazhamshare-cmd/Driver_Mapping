# アプリアイコン

## 必要なファイル

`app_icon.png` - 1024x1024ピクセルのアプリアイコン画像

このファイルを配置後、以下のコマンドを実行してください:

```bash
# アイコンを生成
flutter pub run flutter_launcher_icons

# スプラッシュスクリーンを生成
flutter pub run flutter_native_splash:create
```

## アイコンのデザイン要件

- サイズ: 1024x1024ピクセル
- フォーマット: PNG
- 透過: なし（背景は白）
- デザイン: 鏡文字をモチーフにしたシンプルなデザイン

## 推奨デザイン

鏡文字（逆さまの文字）をシンプルに配置したアイコンがおすすめです。
例: 「鏡」という文字を鏡像反転させたデザイン、またはシンプルな幾何学模様
